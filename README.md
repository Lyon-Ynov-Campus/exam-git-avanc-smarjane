# Site Web en Go

Ce projet est un site web développé en Go qui affiche une page vide pour le moment. 

## Fonctionnalités

Le site affiche pour le moment une page vide. Une fonctionnalité sera développée en parallèle, depuis la branche dev déjà existante.

## Installation

```bash
go mod tidy
go run main.go
```

Le serveur sera accessible sur http://localhost:8080

Differecne entre git reset et revert : 
git reset : Permet d’annuler des commits ou des modifications dans l’historique.
git revert : Annule un commit spécifique en créant un nouveau commit d’annulation.
